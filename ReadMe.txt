Программа принимает на вход массив строк (ввод с клавиатуры). Далее выполняет фильтрацию введенного массива с учетом заданного базиса длины строки.
По умолчанию базис длины равен 3. Т.о. после фильтрации в исходном массиве останутся только те элементы (строки), чья длина меньше 4.
Остальные будут заменены на пустые строки.После этого из отфильтрованного массива формируется новый массив путем удаления всех пустых строк.
